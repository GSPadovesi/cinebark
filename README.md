# cinebark

Cinebark é uma aplicação full stack de cinema criada como projeto de estudos para praticar desenvolvimento backend com Java e Spring Boot integrado a um frontend moderno em React.

O sistema permite listar filmes em cartaz, visualizar detalhes, consultar sessões disponíveis, selecionar assentos e realizar a compra de tickets. Também possui uma área administrativa simplificada para gerenciar filmes, salas e sessões.
